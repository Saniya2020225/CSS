🛒 Amazon Clone

A responsive Amazon Clone built using HTML5 and CSS3. This project recreates the homepage of Amazon with a clean and modern user interface while focusing on responsive web design and front-end development fundamentals.

🚀 Features

- Responsive homepage layout
- Amazon-inspired navigation bar
- Hero banner section
- Product category cards
- Deals and shopping sections
- Footer with multiple links
- Clean and organized code structure
- Beginner-friendly project

🛠️ Tech Stack

- HTML5
- CSS3


🎯 Learning Outcomes

- Semantic HTML
- CSS Flexbox
- CSS Grid
- Responsive Web Design
- Layout Structuring
- UI Design Fundamentals
